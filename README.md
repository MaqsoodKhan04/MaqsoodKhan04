## What I've built

## Cloud Infrastructure Cost Optimization Audit
This repository houses an automated Cloud Infrastructure FinOps Audit Pipeline that programmatically ingests, cleans, and analyzes 90 days of fragmented cloud 
billing logs. Built using a combination of Python (Pandas) for data extraction and normalization and SQL for heavy relational transformations, the core engine 
executes a custom waste-detection algorithm to isolate "Zombie Resources"—expensive compute and storage instances generating costs despite zero utilization. The 
pipeline successfully flags untagged spend, groups infrastructure burn rates by department, and outputs an analytical dataset engineered to reclaim 5.2% in 
monthly spend leakage, translating into a strategic roadmap for over $14,000 in annual OpEx savings.

👉 [View project](https://github.com/MaqsoodKhan04/Cloud-Infrastructure-Cost-Optimization-Audit)

### SaaS Customer Churn Analysis
This is the project I'm most proud of so far. I built a health
scoring system for a SaaS company that identifies which customers
are likely to leave before they actually do.

The health score turned out to actually work — customers in the
red tier churned at 40% while healthy customers churned at only 8.7%.
I presented the findings in a 3 page Power BI dashboard.
